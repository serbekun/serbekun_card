# serbekun.com

Contact card and link page for serbekun.

Live at [serbekun.com](https://serbekun.com).

## What

A minimal terminal-styled landing page. Contains links to socials and projects. No frameworks, no build step, no bullshit.

## Contents

- index.html - page markup
- style.css - terminal theme styles
- 404.html - custom 404 page

## Usage

Open index.html in a browser. That's it.

## Deploy

Static hosting. Drop the files on any server, CDN, or static host. Cloudflare Pages works with zero config.
